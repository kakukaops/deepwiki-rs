# Phase 4: Visualization

**Objective**: Generate Mermaid diagrams.

**Rules**:
1.  **Context**: Use `C4Context` for system boundaries.
2.  **Container**: Use `C4Container` for high-level technology choices.
3.  **Component**: Use `graph TB` or `classDiagram` for internal logic.
4.  **Sequence**: Use `sequenceDiagram` for Workflows.

**Validation**: Ensure syntax is valid and node names do not contain special characters that break rendering.
