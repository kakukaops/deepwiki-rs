# Strategy: Java

## Configuration
- **Extensions**: `.java`
- **Manifest Files**: `pom.xml`, `build.gradle`
- **Ignore Paths**: `target`, `build`, `.gradle`

## Analysis Heuristics
1.  **Module Definition**: Maven/Gradle modules, package structure.
2.  **Interface**: `interface` keywords, Abstract Classes.
3.  **Key Patterns**: Dependency Injection (Spring), Singleton, Factory, Builder.
