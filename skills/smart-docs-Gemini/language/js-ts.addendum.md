# Strategy: TypeScript / JavaScript

## Configuration
- **Extensions**: `.ts`, `.tsx`, `.js`, `.jsx`
- **Manifest Files**: `package.json`, `tsconfig.json`
- **Ignore Paths**: `node_modules`, `dist`, `build`, `.next`

## Analysis Heuristics
1.  **Module Definition**: npm packages, ES Modules (`import/export`).
2.  **Interface**: TypeScript `interface` / `type` definitions.
3.  **Key Patterns**: Hooks (React), Higher-Order Components, Async/Await.
