# Execution Pipeline

To generate documentation, you must execute these phases strictly in order:

1.  **Discovery**: Execute `pipeline/phase-1-discovery.md` (Stack Detection & Strategy Loading)
2.  **Analysis**: Execute `pipeline/phase-2-architecture.md` (Pattern Extraction)
3.  **Drafting**: Execute `pipeline/phase-3-docs.md` (Content Generation)
4.  **Visualization**: Execute `pipeline/phase-4-diagrams.md` (Diagramming)
5.  **Quality Assurance**: Execute `pipeline/phase-5-review.md` (Validation against `compliance/plugin-checklist.md`)
